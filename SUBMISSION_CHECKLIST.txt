                    SUPPORT & TROUBLESHOOTING
===============================================================================

Server Won't Start:
  → Make sure port 9000 is available
  → Try: python -m uvicorn main:app --host 127.0.0.1 --port 9000

API Key Error:
  → Header must be: x-api-key: myhackathonkey123
  → Case-sensitive

Audio Processing Error:
  → File must be valid MP3
  → Try converting with Audacity
  → Check file size (not empty)

Connection Refused:
  → Server not running
  → Start with: python -m uvicorn main:app --host 127.0.0.1 --port 9000

Invalid Language:
  → Must be exactly: Tamil, English, Hindi, Malayalam, Telugu
  → Case-sensitive
